## Week 1-2: Data Cleaning & EDA

### Datasets Used
- Titanic Dataset (891 rows, 12 columns)
- Housing Price Dataset (545 rows, 13 columns)

### Tasks Completed
- Loaded and explored both datasets
- Handled missing values (Age: median, Embarked: mode, Cabin: dropped)
- Removed duplicates
- Performed univariate & bivariate analysis
- Created 11 individual visualizations + 2 comprehensive dashboards
- Generated correlation heatmaps
- Identified key predictors

### Key Insights

**Titanic:**
- Overall survival rate: 38.4% (342 out of 891)
- Women survived at a significantly higher rate than men
- 1st class passengers had highest survival; 3rd class the lowest
- Top predictors: Pclass (-0.34), Fare (0.26)

**Housing:**
- Price range: 1.3M to 13.3M
- Area is the strongest price predictor (0.54)
- Furnished houses command higher prices
- Top predictors: Area, Bathrooms (0.52), Stories (0.42)
